- 概要
	![[circuit_current_measurement.svg]]
	
- 波形発生器 (Function generator)
	- 電圧波形を作る．
	- 主に使う場所
		- square 矩形波
			- 暗い青のボタンで Freq. Ampl, Duty を設定する
			- 電圧は基本 high level で設定．
			- 右の数字キーで値を入力．
		- burst
			- 遅延発生器が int の場合，信号を新しく入力できる．
			- あんまり意味がわかっていない
		- Output
			- 電圧の出力オンオフ
	- 外観
		![[PXL_20260209_101154552.jpg|350]]
- 遅延発生器 (Delay generator)
	- 信号を出す．現状では 1 Hz のパルス波を出力している
	- int 
		- 遅延発生器からトリガ信号を出力する
	- ext 外部からトリガーを入力する．
		- 左下の EXT TRIG にカメラからのトリガ信号が入力される
		![[PXL_20260209_101205103.jpg|350]]
	- AB, CD に信号を分岐させる．ひとつはADコンバータ，ひとつはファンクションジェネレータへ．
- カメラ
	- 撮影開始と同時にトリガー信号を出力できる
	- TRIG TTL OUT から出てくる．同軸ケーブルで接続．BNC 
- フォトカプラ (photo-coupler)
	- カメラからのトリガ信号と回路を電気的に絶縁するために導入．
	- 効果があるかはなんとも
	- 電源は，スイッチを右側がON，左側がOFF
		![[photo-coupler.png|300]]
- [[電圧フォロワ]] (Voltage follower)
	- 信号の電圧波形を負荷 (水電解セル) に伝える．回路には[[オペアンプ]]を使用
		![[voltage_follower.png|300]]
	- MEMO
		- 信号側のインピーダンスが大きく，負荷のインピーダンスが小さい場合，電圧降下で負荷にかかる電圧が小さくなってしまう → 1倍の増幅回路で，負荷側の電圧を固定する．
- [[電流計測回路]]
	- 電源は，スイッチを右側がON，左側がOFF
	- **結構電池がなくなりがち**なので，適宜充電する．
	- 測定前に 1k Ohm, 1 V の条件で電流を計測し，正しい結果が出力されることを確認
	- 
		![[current_measurement.png|300]]

- ADコンバータ
	- アノード電圧，カソード電圧，電流計測回路の出力電圧を計測．
